# SwampTimers Add-on

Advanced timer scheduling for Home Assistant.

## About

SwampTimers provides flexible timer scheduling with support for:
- Duration-based timers (start time + duration)
- Time range timers (on time to off time)
- Day-of-week filtering
- Multiple storage backends (YAML or SQLite)

## Installation

1. Add this repository to your Home Assistant add-on store
2. Install the SwampTimers add-on
3. Configure your preferred settings
4. Start the add-on
5. Access via the sidebar panel

## Configuration

See the Configuration tab in the add-on for available options.
